## 用导出的数据进行导入

上一章节讲了如何导出。导入则是使用导出的数据。

方式是进入到 `Manage Sources` 命令，对任意阅读源唤起 Actions，进行如下操作：

![batch_import_sources_action](https://tidyread.s3.us-west-2.amazonaws.com/batch_import_sources_action.png)

拷贝导出的数据到文本框后，`cmd + Enter` 进行保存：
![batch_import_sources](https://tidyread.s3.us-west-2.amazonaws.com/batch_import_sources.png)

<Callout type="info">
    校验会花一些时间（主要是RSS Link校验耗时），请耐心等待。
</Callout>
<Callout type="warning">
    若导入失败，请参考[文档](https://www.tidyread.info/zh-CN/docs/why-digest-failed#网络屏蔽)进行排查。
</Callout>

## 直接手写导入

正常情况下录入阅读源是通过 `Add Source` 表单。如果你嫌麻烦，也想批量导入，可以参考以下类型，在文本框中填写的内容符合 `Source[]`（需要是一个数组） 即可：

```typescript
export interface Source {
    url: string;
    title: string;
    schedule: "everyday" | "custom";
    customDays?: (
        | "Mondy"
        | "Tuesday"
        | "Wednesday"
        | "Thursday"
        | "Friday"
        | "Saturday"
        | "Sunday"
    )[];
    rssLink?: string;
    // Time span of the content pulled by rss, default is 1, means one day
    timeSpan?: string;
    tags: string[];
    // favicon url
    favicon?: string;
}
```
