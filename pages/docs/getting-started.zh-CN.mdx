## 前提

Tidyread 目前只提供了 Raycast 插件，如果你还没有安装 Raycast ，强烈推荐你试试。可以在[这里](https://raycast.com/?utm_source=tidyread)下载。

## 安装

安装 [Tidyread Raycast 插件](https://www.raycast.com/jaredliu233/tidyread---streamline-your-daily-reading)

## 启动

<Callout type="warning">
    中国用户建议先配置 Http
    Proxy（需科学上网），否则后续拉取RSS（用于生成简报）的过程可能会由于哔哔原因而失败。
</Callout>

打开 Raycast 设置页面找到 `Http Proxy` 配置项，填入科学上网工具的 http proxy：
![http_proxy](https://tidyread.s3.us-west-2.amazonaws.com/http_proxy.png)

注意以上端口号部分，每个科学上网工具是不同的，需要自行查看。比如我的是从下图位置导出的：
![tz_http_proxy](https://tidyread.s3.us-west-2.amazonaws.com/tz_http_proxy.png)

接下来从 Raycast 进入到 `Manage Source List` 命令，为了帮助用户更快上手，你会看到一个启动引导：
![onboard_guide](https://tidyread.s3.us-west-2.amazonaws.com/onboard_guide.png)

选择你感兴趣领域提交后，Tidyread 会自动为你添加一些阅读源，并且跳转到 `Daily Read` 命令（你最常用的命令）自动生成一份简报：
![digest_generating](https://tidyread.s3.us-west-2.amazonaws.com/digest_generating.png)

如果你不喜欢推荐的阅读源，可以跳过这个步骤：
![skip_onboard_guide](https://tidyread.s3.us-west-2.amazonaws.com/skip_onboard_guide.png)

## 添加你的第一个阅读源

如果你跳过了引导，则需要手动添加阅读源，你会看到一个空列表：

![empty_list](https://tidyread.s3.us-west-2.amazonaws.com/tidyread_empty_list.png)

点按 Enter 会触发创建阅读源：

![add_source](https://tidyread.s3.us-west-2.amazonaws.com/tidyread_add_source.png)

如果经过引导，会自动创建一些阅读源，那么你在`Manage Source List`中可以自行通过`cmd + N` 创建订阅源，仍然会进入到上图的界面。

只有 URL 和 Title 是必填项，其他的都是可选的。
通过 Schedule 字段，你可以规划该阅读源的阅读时间，它会出现在那一天的 `Daily Read` 中。

<Callout type="info">
    只有填写了 RSS
    Link，阅读源才能用于生成简报。如果你想了解如何找到网站的RSS，可参考[文档](/zh-CN/docs/where-to-find-rss)。
</Callout>
<Callout type="warning">
    如果 `RSS Link` 一直校验超时导致无法保存，可能是因为没有配置 Http
    Proxy（需科学上网），参考[启动](#启动)。
</Callout>

## 尝试生成简报

进入 `Daily Read` 命令， 如果跳过了启动引导，可以看到简报尚未生成：

![daily_read_empty_digest](https://tidyread.s3.us-west-2.amazonaws.com/daily_read_empty_digest.png)

点按 Enter 键，等待简报生成：
![daily_read_has_digest](https://tidyread.s3.us-west-2.amazonaws.com/daily_read_has_digest.png)

生成完毕后，你可以在 `Daily Read` 中看到简报。点击 Enter 键，可以查看简报的详情:
![todays_digest](https://tidyread.s3.us-west-2.amazonaws.com/todays_digest.png)

由于目前没有设置 AI 相关的配置，所以简报中的 所有内容都是原始内容，并没有经过 AI 摘要。

到此为止，你已经用到了 Tidyread 的基础功能，如果你希望进一步提升阅读体验，请继续往下看。
